# video-calling-app
## Tech stack
1. Node.js 19.6
2. React.js 18.2
3. socket.io
4. WebRTC

## packages
1. react-player: To play video on application
2. react-router-dom: Navigate in the application
3. nodemon: Helps with the speedy development of Node. js applications.

## Cloning the repository

```shell
git clone https://github.com/santoshkamble8168/video-calling-app
```

### Start frontend

```shell
cd client
```

### Install packages

```shell
npm i
```

### run the frontend

```shell
npm start
```

### Start backend

```shell
cd server
```

### Install packages

```shell
npm i
```

### run the backend

```shell
npm run dev
```
